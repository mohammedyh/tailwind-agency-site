## Run Locally

Clone the project

```bash
git clone https://github.com/mohammedyh/tailwind-agency-site.git
```

Go to the project directory

```bash
cd tailwind-agency-site
```

Install dependencies

```bash
pnpm install
```

Start the server

```bash
pnpm dev
```
